# 🚢 ShipDeckKK

ShipDeckKK is a responsive company profile website designed for a premium maritime business specializing in Luxury Yachts, Cargo Ships, and High-End Liners.  

## ✨ Features
- Home page with company highlights
- Gallery page displaying ship collections
- About page for company background
- Services page showcasing offerings
- Subscription page for customer engagement

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎨 Design & Prototyping
- Designed using Figma before development
